# New app
## Project setup
```
yarn install
```
Also rename .env.example to .env.local and set VUE_APP_NEWS_API_KEY to your newsapi.org api key

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
